# ­ЪЇ» рЃАрЃЮрЃфрЃўрЃљрЃџрЃБрЃарЃў рЃЏрЃћрЃЊрЃўрЃўрЃА рЃЏрЃљрЃарЃЎрЃћрЃбрЃўрЃюрЃњрЃўрЃА рЃљрЃњрЃћрЃюрЃбрЃў
# Social Media Marketing Agent for Organic Honey

рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃЏрЃљрЃарЃЎрЃћрЃбрЃўрЃюрЃњрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃЮрЃарЃњрЃљрЃюрЃБрЃџрЃў рЃФрЃЏрЃарЃўрЃА рЃЉрЃўрЃќрЃюрЃћрЃАрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃДрЃћрЃюрЃћрЃЉрЃА **рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ Google-рЃўрЃА рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃА**:

- ­ЪЊЮ **рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ**: Google Gemini API (рЃЦрЃљрЃарЃЌрЃБрЃџ рЃћрЃюрЃљрЃќрЃћ)
- ­Ъје **рЃАрЃБрЃарЃљрЃЌрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ**: Google Vertex AI (Imagen)
- ­Ъцќ **Telegram Bot**: рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃў рЃЊрЃљрЃЏрЃбрЃЎрЃўрЃфрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў
- ­Ъїљ **Streamlit Dashboard**: рЃЋрЃћрЃЉ рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА

## Рюе рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў

### ­Ъцќ Google Gemini рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ
- рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў Facebook рЃърЃЮрЃАрЃбрЃћрЃЉрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ **рЃЦрЃљрЃарЃЌрЃБрЃџ рЃћрЃюрЃљрЃќрЃћ**
- рЃЏрЃЮрЃарЃњрЃћрЃЉрЃБрЃџрЃў рЃбрЃЮрЃюрЃў рЃЊрЃљ рЃАрЃбрЃўрЃџрЃў
- рЃФрЃЏрЃарЃўрЃА рЃАрЃљрЃ«рЃћрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ

### ­Ъје Google Vertex AI (Imagen)
- рЃърЃарЃЮрЃцрЃћрЃАрЃўрЃЮрЃюрЃљрЃџрЃБрЃарЃў рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃўрЃА рЃцрЃЮрЃбрЃЮрЃћрЃЉрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ
- рЃарЃћрЃљрЃџрЃўрЃАрЃбрЃБрЃарЃў, рЃЏрЃљрЃдрЃљрЃџрЃў рЃ«рЃљрЃарЃўрЃАрЃ«рЃўрЃА рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃў
- рЃЏрЃЮрЃарЃњрЃћрЃЉрЃБрЃџрЃў рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃў рЃАрЃбрЃўрЃџрЃў

### ­ЪЊ▒ Telegram Bot
- `/create [рЃФрЃЏрЃарЃўрЃА рЃбрЃўрЃърЃў]` - рЃљрЃ«рЃљрЃџрЃў рЃърЃЮрЃАрЃбрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
- Inline рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃў:
  - РюЁ рЃЊрЃљрЃЊрЃљрЃАрЃбрЃБрЃарЃћрЃЉрЃљ
  - ­Ъћё рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљ
  - ­Ъје рЃцрЃЮрЃбрЃЮрЃА рЃерЃћрЃфрЃЋрЃџрЃљ
  - РюЈ№ИЈ рЃЊрЃљрЃерЃЉрЃЮрЃарЃЊрЃќрЃћ рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ
  - РЮї рЃБрЃљрЃарЃДрЃЮрЃцрЃљ

### ­Ъїљ Streamlit Dashboard
- рЃЋрЃћрЃЉ рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў рЃЊрЃарЃљрЃцрЃбрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА
- рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃ«рЃћрЃџрЃўрЃЌ рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ
- рЃАрЃбрЃљрЃбрЃўрЃАрЃбрЃўрЃЎрЃљ рЃЊрЃљ рЃљрЃюрЃљрЃџрЃўрЃбрЃўрЃЎрЃљ
- рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃўрЃАрЃбрЃЮрЃарЃўрЃљ
